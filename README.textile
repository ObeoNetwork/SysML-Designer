h1. Welcome to SysML Modeling by Obeo Network

This modeler provides a set of common diagrams to work with SysML models.

These plugins are released under the EPL Open-Souce License.

More details about features, screenshots, release notes, dependencies here: http://www.umldesigner.org/ref-doc/presentation.html#sysml-designer

This modeling tool has been created with: http://www.eclipse.org/sirius

To build the module you'll need Maven3 and type @mvn clean verify checkstyle:checkstyle@
